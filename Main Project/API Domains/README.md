# API Domains

This folder contains all real-time API-based smart city domains.

## Included Domains

- Pollution API
- Public Safety API
- Traffic  API

Each domain contains:
- Python API extraction scripts
- SQL database scripts
- Dashboard files
- Processed datasets
- Output visualizations

## Workflow

API Data
   ↓
Python Processing
   ↓
SQL Storage
   ↓
Dashboard Visualization
