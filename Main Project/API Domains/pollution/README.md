# Pollution API Domain

## Overview

This module collects and analyzes real-time air pollution data using APIs.

## Technologies Used

- Python
- REST API
- SQL
- Power BI
- Streamlit

## Features

- Real-time AQI monitoring
- PM2.5 and PM10 analysis
- Pollution trend visualization
- City-wise air quality analysis

## Folder Structure

- Pollution_API_Codes
- Pollution_SQL_Database
- Pollution_Dashboard
- Pollution_Data

## Workflow

Pollution API
   ↓
Python Extraction
   ↓
SQL Database
   ↓
Dashboard Analytics
