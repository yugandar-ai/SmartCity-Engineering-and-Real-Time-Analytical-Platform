# Bangalore City Traffic Analysis

## Overview

This module analyzes Bangalore city traffic accident and congestion data using CSV datasets.

## Features

- Traffic accident analysis
- Congestion trend analysis
- Year-wise accident statistics
- Traffic dashboard visualization

## Technologies Used

- Power BI
