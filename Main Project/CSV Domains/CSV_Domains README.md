# CSV Domains

This folder contains historical smart city datasets collected from CSV files.

## Included Domains

- Bangalore City Traffic
- Chennai Water Consumption
- Hyderabad Public Safety
- Mumbai Electricity Consumption

## Features

- Historical data analysis
- SQL data storage
- Dashboard visualization
- Machine learning analytics

## Workflow

CSV Dataset
     ↓
Python Cleaning
     ↓
SQL Database
     ↓
Analytics Dashboard
